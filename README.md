# Image-Mania
Cloned an image gallery web application using Pexel API in HTML, CSS, and JavaScript where you can see stock images in real-time. Users can also search for a particular image in the search box and can easily see the preview and download it at HD quality.

**Description**

The Pexel Clone project is a web application that provides users with a platform to search and view images. It replicates the functionalities of Pexel, a popular online photo-sharing platform. Users can search for images using keywords and browse through the search results. The application displays the images in a gallery format, allowing users to preview and download the images of their choice.

**Installation**

To run the Pexel Clone project locally, follow these steps:

1. Clone the repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory.

   ```bash
   cd image-mania
   ```

3. Open the index.html file in your preferred web browser.



**Usage**
1. Open the application in your web browser.
2. The homepage displays a search section with a background image and a search box.
3. Enter a keyword in the search box to search for images related to that keyword.
4. Press the Enter key to initiate the search.
5. The application will display the search results in a gallery format.
6. Click on an image in the gallery to open a lightbox view with a larger preview of the image.
7. In the lightbox view, you can see the name of the photographer and download the image by clicking the download button.
8. To load more images, click the "Load More" button at the bottom of the page.
9. Enjoy exploring and downloading images!

**Features**
1. Search for images using keywords
2. Display search results in a gallery format
3. Preview images in a lightbox view
4. View photographer information
5. Download images
